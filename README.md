# MEEP Builder
A browser-based GUI for building MEEP photonics simulation scripts.

## Features
- 2D/3D geometry builder with live preview
- Source, monitor, boundary condition setup
- Auto-generates Python/MEEP script
- Periodic arrays, T/R analysis, scatter mode

## How to use
Download and open it in any browser — no installation needed.
